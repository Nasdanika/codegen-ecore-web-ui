Root Configuration

Target Project

Target project to output generated artifacts to. If blank, then the project which contains the model is used as the target project.

Source Folder

Source folder for Java classes and classloader resources.

Base Package

Base package. Generated classes and resources are placed into <base package>.<epackage path> package where epackage path is EPackage name for top level EPackage's and dot-separated list of names for nested EPackage's.

Description

Model description, not used for generation.
